## Clone this project

```bash
> git clone https://github.com/LitRHap/self-wa
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd self-wa
> npm i
```

### Usage
```bash
> npm start
```
